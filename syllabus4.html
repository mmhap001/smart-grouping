Course Description

This Data Mining course is about all about going deep with the data. Search for data to it's fullest reaching down it lowest level

Data Mining as the words say, we need to mine the data in order to get all the information. In this course, there will

be 4 modules. Each module will cover exactly 3 chapters from the textbook. There will be 5 assignments, each of 100 marks.

Students will learn the basics of Data Mining. There will be challenges during the course which students might find it difficult to cope.

There will be a chance that the course will be curved.
